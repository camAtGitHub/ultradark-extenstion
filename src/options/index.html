UltraDark Options

Developer Settings

When enabled, detailed debug messages will appear in the browser console.

Scheduling

During the window, UltraDark toggles on automatically.

Regex Exclusions

Add one pattern per line. Accepts plain strings or /regex/flags.

Per-Site Overrides

Configure UltraDark behavior for specific sites. Changes are saved automatically.

Reset Settings

Reset all settings to their default values. This will clear all per-site overrides, regex exclusions, and restore default theme settings.
